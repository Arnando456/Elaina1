## Clone this project

```bash
> git clone https://github.com/Arnando456/Elaina1
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd Elaina1
> bash install.sh
```

### Usage
```bash
> npm start
```
